Library
Morisset Library is a great location for the silent studier. Four silent floors individual cubicle desks along with one floor with access to computers allows for a perfect spot for all kinds of studiers. The library also has group study rooms on select floors available for reservation on the uOttawa library website. As an added bonus, there is a second cup on the first floor for those high functioning caffeine addicts.
